gnatt
=====

An [MQTT-SN](http://mqtt.org/new/wp-content/uploads/2009/06/MQTT-SN_spec_v1.2.pdf) implementation in Go

Including
---------
- MQTT-SN Gateway (in progress)
- MQTT-SN Forwarder (in the future)
- MQTT-SN Client Library (in the future)
